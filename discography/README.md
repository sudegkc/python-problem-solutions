Project: Record Company Database Manager

This Python script manages a multi-file database for a record company, processing interconnected artist and song data.

Key Features:

Multi-level File I/O: Parses a master artists.txt to dynamically access individual artist files.

Data Aggregation: Uses a dictionary-based structure to group and sort songs by recording year.

Chronological Output: Prints all recordings in ascending order, paired with their respective artist codes.

File Structure:

artists.txt: Master list of artist codes and filenames.

*.txt: Artist-specific files with years and song titles.

solution.py: Main logic for data processing and sorting.
